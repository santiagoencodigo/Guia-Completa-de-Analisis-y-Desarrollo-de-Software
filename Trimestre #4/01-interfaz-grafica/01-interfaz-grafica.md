## Introducción

Se estará viendo etiquetado HTML-CSS Básico.

> Se recomienda el cuidado de los archivos y por ende un mockups.

¿Qué es una interfaz gráfica?

Cuando el usuario tiene una relacion con un sistema, la interfaz es lo que permite visualizar e interactuar con el sistema.

---

El tecnologo pide esto:

* Página Web = HTML + CSS + JS + PHP + MySQL

Solo el apartado de diseño es HTML y CSS como tecnologías web.

Solamente HTML y CSS es estaticas, pues cuando es dinamico es cuando guarda y procesa sistemas.
